Country Finder

Single Player Online Map based Game

Country Finder is a game that challenges users to test their ability to state the location of world countries in a fun and adventurous way. Easy level will contain a very high probabilty of countries that are widely popular and few random countries, Intermediate level will have some popular countries and rest random countries while in hard level all of the countries will be selected at random.

Instructions

1. The user will be enquired about 10 countries individually and user has to guess it's location.

2. User can make 3 attempts for each country.

3. User will be provided 40 sec to correctly select the answer.

4. T => Time Left, P=> Points Earned, Q=> Questions Left, A=> Attempts Left

5. Red circles will appear after wrong attempts that will narrow down the required country location on map.
